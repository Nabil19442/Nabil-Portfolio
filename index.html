Hospital Management System

C Stack Queue Tree Binary Tree AVL Tree

A comprehensive console-based Hospital Management System developed using C programming language, focusing on real-world data structure applications and efficient patient handling. This project simulates how hospitals manage normal patients, emergency cases, doctors, and staff using optimized algorithms and structured data storage. The system prioritizes emergency patients using a Max-Heap, ensuring critical cases are handled first, while normal patients are managed through a queue (FIFO). Doctors are stored and managed using a Binary Search Tree (BST) for fast searching, insertion, and deletion. Hospital staff records are maintained using an AVL Tree, guaranteeing balanced and efficient operations.
